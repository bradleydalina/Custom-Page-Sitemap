This plugin generate a custom sitemap with search field, page, categories, archives and recent posts

== Description ==

Custom Page Sitemap plugin simply allows you to output pages links, and other custom links in your website. And does not need you to manually insert code and function to generate links. It can generate Pages, Categories, Archives and Recent Posts at once.
You can add a limited icon though it only accepts for now html entity unicode characters. You can call it more than twice or more to generate specific targets. Like specific categories in Separate sections or element.

"This plugin is limited with its available settings. Before writing a reviews, please *mention that you read the whole description* and clearly understand the limit and usage of the plugin."

== Installation ==

This section describes how to install the plugin and get it working.

e.g.
1. Upload the entire `custom-page-sitemap` folder to the `/wp-content/plugins/` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Goto page menu and select at the submenu links Custom Page Sitemap for the plugin settings and informations.

== Frequently Asked Questions ==

= What does it do?  =

It add page links, merge with archives, categories, recent posts and search box option.

= What makes it different with other sitemap plugins

It has more available options, it can sort the links, and it can also append custom links it also allows you to set your own alias to the shortcode.


= What are its limitations? =

This plugin may not meet all the expected output you need according to your needs, but it is still open for improvements.
